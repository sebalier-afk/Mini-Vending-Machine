# Mini-Vending-Machine
This is my first prodject a 24-inch-tall vending machine that uses a Razberry pi and 6 servos to dispense drinks and snacks.

🤔Why
I have been wanting to build a hardware prodject for a while now and I love vending machines. I think its so cool how vending machines automatically dispense objects and you can just pick it up no human interation required.

✨Features
* 6 slots that can each hold like 5 items for a total of 30 items stored.
* Servos that automatically reset once all items have been vended
* 6 buttons one for each slot
* A acrylic window to look at the snacks through
* Slideupable side so you can refill your snacks
